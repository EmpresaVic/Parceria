💾 Configurar Pasta de Backup

Em navegadores compatíveis (Chrome/Edge), você pode escolher uma pasta fixa do computador para onde os arquivos de backup serão salvos automaticamente. Se o navegador não suportar essa função, o backup continuará funcionando normalmente via download do arquivo .json.

⬆️ Importar Backup de Outra Máquina

A importação é aditiva e segura: apenas os registros que ainda não existem neste computador serão adicionados. Nada que já está cadastrado aqui será apagado, sobrescrito ou alterado.

🔗 Sincronização com GitHub

Os dados continuam salvos instantaneamente neste computador. Com a sincronização configurada, o sistema também envia automaticamente uma cópia para um repositório do GitHub, alguns segundos depois de cada alteração — sem precisar clicar em nada. As fotos dos pratos são enviadas como arquivos separados (não ficam no JSON).

⚠️ O token é salvo apenas neste navegador (nunca é enviado ao repositório). Como o repositório é público, qualquer pessoa com este token teria acesso de escrita — guarde-o com o mesmo cuidado de uma senha, e revogue no GitHub se este computador for compartilhado.

🔗 Mesclar Cadastros Duplicados

Use isto quando a mesma pessoa foi cadastrada duas vezes por engano (ex.: em setores diferentes). Escolha qual cadastro é o correto (vai ficar) e qual é o duplicado (vai ser removido). Todos os pedidos do duplicado — inclusive os já registrados — passam a ficar com o nome e o setor do cadastro correto.
